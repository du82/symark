SyMark: Transform Your SiYuan Notes into Beautiful Websites

May 6th, 2025

SyMark is a privacy-focused, Rust-based static site generator that converts SiYuan notebooks into responsive websites with a single terminal command. Every aspect of your notes—bidirectional links, tags, and all formatting options—renders elegantly on the web with no manual adjustments needed.

As a demonstration of SyNote, this very website is generated with SyNote, using this SY file as the input content.

Lightning-Fast Generation

SyMark processes even large notebooks (tested wtih 500k notes) in seconds, generating optimized HTML files ready for deployment anywhere.

Preserve Your Work

Every aspect of your notes—from complex diagrams to LaTeX equations—renders perfectly on the web with no manual adjustments needed.

Zero Configuration

See How to use SyMark for more info.

Customizable Themes

Choose from built-in themes or create your own with straightforward CSS customization options.

Privacy Control

Easily exclude private notes with configurable filters based on tags, attributes, or folder paths.
